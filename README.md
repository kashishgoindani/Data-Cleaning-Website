# Data Cleaning Website

A full-stack data cleaning web application that allows users to upload datasets, clean missing values, remove duplicates, handle outliers, and preprocess data efficiently.

## Features
- Upload CSV and Excel files
- Auto detect column types (Numeric, Categorical, Date)
- Handle missing values (Remove, Mean, Median, Mode)
- Remove duplicate rows
- Download cleaned file

## Tech Stack
- Frontend: React
- Backend: Flask (Python)
- Database: MongoDB Atlas
- Deployment: Vercel (Frontend) + Railway (Backend)
